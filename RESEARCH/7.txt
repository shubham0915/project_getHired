My approach to Entity masking for PII with Local models
This is a project done for a major bank in Europe. For what it seems, they (and other banks) have been adopting the use of 70B models for all proposes, instead of several models that really don't fit several use cases.

The motivation here is the following: Using SOTA LLMs with paramount data protection.

The Banks want to use Claude 3.5 and GPT-4 with RAG, so PII and EM needs to be addressed. So i did this small use case that i would like to share with you, that was built on top of Presidio from Microsoft.

Size matters:

We tested several models, present in AWS and Azure, to avoid quantization from Ollama for example, to see how it performs.
We tried several methods from CoT to ReAct for small models, but they never make the cut (100%). We saw that models like Phi3.5 and llama 11B were good surprises, but the problem is always on the unstructured data (e.g. Weird InvoiceId spaced somewhere).

r/LocalLLaMA - My approach to Entity masking for PII with Local models 
Given the size of the model, its done in two requests, list PII and return a dictionary with the data. The 70b model is quantized Fp16 and Fp8 and still works just fine.

So if you are in the EU, this approach is GDPR proof. You cannot use LLama 3.2 multi modal for example, so some are using Qwen 2.5 (THE IRONY).

Presidio is only used a last layer of security, that barely have any hits.

We tried this as well with some internal tooling and came to very much the same conclusion: larger models pick up more data. Structured data like JSON performed worse than prose-text. Anything below 8B was useless.

Our difference was that we were mainly working with structured data from known tools, so we could do NER analysis on a field-based level, which worked pretty well, and NER-frameworks performed just as well as an LLM (better in some cases because no hallucinations).

We ended up with a two-level solution: NER for an initial filter, then a "hard" filter that detects and blocks everything else in case something slips through.

Still not 100% though.
